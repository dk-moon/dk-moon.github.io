2024.01-2024.06

강건한 혼합 효과 신경망을 이용한 시선 추정

Github Paper
단국대학교 학위논문

시선 추정 모델은 사용자별 생리적 차이와 환경 변화(조명, 머리 자세)로 인해 정확도가 저하되는 문제가 있습니다. 이를 해결하기 위해 강건한 혼합 효과 신경망(Robust Mixed-Effects Neural Network, RMEN)을 도입하였습니다.

이 모델은 고정 효과(Fixed Effects)와 랜덤 효과(Random Effects)를 함께 학습하여 개인별 특성을 반영합니다. 기존 모델이 모든 사용자에게 동일한 가중치를 적용하는 반면, RMEN은 사용자별 임베딩을 학습하여 맞춤형 시선 추정을 수행합니다.

또한, Adversarial Training과 Bayesian Regularization을 활용하여 환경 변화에도 강건한 성능을 유지하며, 다중 학습(Multi-Task Learning) 기법을 적용하여 머리 자세 추정과 병행 학습을 수행하였습니다. 실험 결과, 기존 모델 대비 각도 오차(Angular Error)를 15~20% 감소시키며, 새로운 사용자에게도 빠르게 적응하는 성능을 확인하였습니다. 향후 연구로는 Few-shot Learning 및 Edge AI 경량화를 통해 실시간 적용 가능성을 높이고자 합니다. 🚀

Github Paper
단국대학교, 한국생산기술연구원, PNS빅데이터과학연구소

열처리 공정에서는 제품 배치에 따라 온도 분포가 달라져 품질에 큰 영향을 미칩니다. 이를 해결하기 위해 선형 회귀 및 객체 검출 기반 제품 배치 최적화 모델을 연구하였습니다.

먼저, 열처리로 내부의 온도 변화를 예측하기 위해 선형 회귀 모델을 적용하여, 제품의 위치와 온도 분포 간의 상관관계를 분석하였습니다. 이후, 객체 검출 모델을 활용하여 공정 내 제품 배치를 자동 인식하고 최적화된 배치 패턴을 도출하였습니다.

또한, 실제 공정 데이터를 기반으로 모델을 학습하고, 온도 편차를 줄이는 최적 배치를 도출하였습니다. 실험 결과, 기존 배치 방식 대비 제품 간 온도 편차가 10% 이상 감소하며, 품질 향상과 공정 효율 개선 효과를 확인할 수 있었습니다.

향후 연구로는 강화학습을 적용하여 최적 배치를 실시간으로 추천하는 AI 시스템 개발을 목표로 하고 있습니다. 🚀
