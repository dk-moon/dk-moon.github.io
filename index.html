👀 강건한 혼합 효과 신경망을 이용한 시선 추정

Github Paper
단국대학교 학위논문

🔍 문제 정의
기존 시선 추정 모델은 사용자별 생리적 차이와 환경 변화(조명, 머리 자세 등)에 취약하여 정확도가 저하되는 문제가 있었습니다. 특히, 특정 사용자 그룹에서는 모델의 성능 편차가 커 실환경 적용이 어렵다는 한계가 있었습니다. 이를 해결하기 위해 **강건한 혼합 효과 신경망(Robust Mixed-Effects Neural Network, RMEN)**을 연구하였습니다.

🧠 모델 설계 및 접근 방식
기존 모델이 모든 사용자에게 동일한 가중치를 적용하는 반면, RMEN은 고정 효과(Fixed Effects)와 랜덤 효과(Random Effects)를 함께 학습하여 개인별 특성을 반영합니다. 이를 위해 🎯 사용자별 임베딩을 학습하는 서브네트워크를 추가하고, 맞춤형 시선 추정을 수행하도록 개선하였습니다.

⚙️ 모델 강건성 개선
또한, Adversarial Training 기법을 적용하여 특정 환경에 과적합되지 않도록 학습하며, Bayesian Regularization을 활용하여 새로운 사용자 환경에서도 안정적인 성능을 유지하도록 설계하였습니다.

📊 실험 결과
테스트 결과, 기존 모델 대비 각도 오차(Angular Error)가 15~20% 감소하였으며, 다양한 조명과 머리 자세에서도 일관된 성능을 유지하는 것을 확인하였습니다.

🚀 향후 연구 방향
향후 연구로는 Few-shot Learning을 적용하여 소량의 데이터만으로 새로운 사용자에게 적응할 수 있는 AI 모델을 개발하고, Edge AI 최적화를 통해 경량화된 실시간 시선 추정 시스템 구축을 목표로 하고 있습니다. 🤖📡
